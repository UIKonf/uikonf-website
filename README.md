# UIKonf website

## Development

Check out the repository on your computer, then in the Terminal run:

```bash
bundle install
bundle exec jekyll serve
```

In the browser, open:

```bash
http://127.0.0.1:4000
```

## Deployment

### Using git

- Edit files
- Push to the `gh-pages` branch
- Wait five minutes for the site to build and propagate
